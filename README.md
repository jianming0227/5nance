# Bootstrap Frontend + MongoDB Project

## Tech Stack
- HTML, CSS, JavaScript, Bootstrap
- Node.js + Express
- MongoDB Atlas

## Getting Started
### Frontend
- Go to `/frontend`
- Open `index.html` in a browser

### Backend
- Go to `/backend`
- Run `npm install`
- Create a `.env` with your MongoDB URI
- Run `node server.js`
